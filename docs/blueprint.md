# **App Name**: ARCO Factory Manager

## Core Features:

- Secure Authentication: Email & Password login with admin role authorization via Firebase Authentication.
- Real-time Dashboard: Display total sales, expenses, stock, and profit/loss using data from Cloud Firestore.
- Inventory Management: Add, update, and delete items with categories (Aluminium, Glass, Accessories) and automatic stock updates based on sales.
- Customer Directory: Store and manage customer information (name, phone, address).
- Sales Transactions: Process sales transactions, automatically reduce stock, calculate total, and store sales history in Cloud Firestore.
- Expense Tracker: Record and categorize expenses (Labour, Transport, Electricity, Other) with date and amount.
- Dynamic Reporting: Generate daily, monthly, and profit & loss reports from Cloud Firestore data.

## Style Guidelines:

- Primary color: Dark Blue (#1A237E) to convey professionalism and stability.
- Background color: Light Grey (#F5F5F5) for a clean, factory-friendly look.
- Accent color: White (#FFFFFF) for contrast and highlighting key information.
- Body and headline font: 'PT Sans', a humanist sans-serif for readability and a modern feel.
- Use clear, industrial-themed icons to represent items, categories, and actions.
- Maintain a simple, intuitive layout optimized for ease of use in a factory environment.
- Use subtle loading indicators and transitions to enhance the user experience.
